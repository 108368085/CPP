第一次添加前開WindoewsCMD，照打以下指令 :
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/108368085/cpp.git
git push -u origin main

之後添加打以下指令打在WindoewsCMD : 
cd C:\Users\user\Cpp
git add .
git status
git commit -m "Push"
git push -u origin main

刪除檔案教學:
git rm <namefile.txt>
git status
git commit -m "Remove jpg"
